Project 3 - Social Web

Role: Full Stack Developer

Project Description

Developed a social media application using Django, designed to connect users and enable them to share images with captions. The application focuses on delivering a seamless user experience with real-time interaction capabilities and a responsive design.

Responsibilities

  • User Authentication and Authorization: Implemented secure user login, registration, and profile management using Django's built-in authentication system.
  • Database Design and Management: Structured and managed the database to store user data, posts, and interactions efficiently.
  • User Interface Development: Designed and implemented a responsive UI using Django templates, HTML, and CSS to ensure the application is accessible across various devices.
  • Image Upload Feature: Developed functionality for users to upload images with captions, allowing them to share media content on their profiles.
  • Testing and Debugging: Conducted thorough testing and debugging to ensure smooth, secure, and error-free functionality across the application.

Technologies Used

Django, HTML, CSS, JavaScript
