Project 3 - Social Web

Overview

Developed a social media application that allows users to connect, share images with captions, and interact with each other. The platform features a responsive design, ensuring a seamless user experience across various devices.

Features

  • Integrated user authentication and authorization using Django.
  • Designed and developed a responsive UI using Django templates and CSS.
  • Stored and managed user data in a database.
  • Users can upload images with captions, which are displayed in their profiles.

Technologies Used

Django, HTML, CSS, JavaScript
